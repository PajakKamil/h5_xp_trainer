# Heroes 5 XP Trainer

### Opis działania
Trainer **nadpisuje zdobywanie XP przez WSZYSTKIE postacie** w grze. 

Modyfikacja polega na przechwyceniu bazowej metody gry odpowiedzialnej za przyznawanie doświadczenia. Zamiast standardowej wartości, każdemu bohaterowi (zarówno Twojemu, jak i przeciwnikowi) dodawane jest **+2 000 000 000 (2 miliardy)** punktów doświadczenia przy najbliższej okazji.

---

### Instrukcja obsługi

1. **Uruchom grę**, a następnie ten program (jako Administrator).
2. **Aktywacja:** Kliknij klawisz **[F2]** w dowolnym momencie.
3. **Zdobywanie XP:** Wykonaj w grze dowolną akcję dającą punkty doświadczenia (np. walka, otwarcie skrzyni).
4. **Dezaktywacja:** Ponownie kliknij klawisz **[F2]**, aby wyłączyć modyfikację i przywrócić normalne zdobywanie XP.

---

### Ważne informacje
*   **Wszyscy zyskują:** Pamiętaj, że patch działa globalnie – jeśli przeciwnik zdobędzie doświadczenie, gdy trainer jest aktywny, on również otrzyma bonus.
*   **Uprawnienia:** Program wymaga uprawnień administratora do modyfikacji pamięci procesu gry.
